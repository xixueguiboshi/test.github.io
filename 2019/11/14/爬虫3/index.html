2.4函数

2.4.1何为函数？

是一套定义好的流程，可以包含输入和输出，比如输入数据，返回结果等，两个函数之间可能相互独立的，也可能一个函数输入是另一个函数的输出，也可能是一个函数内部调用另一个函数。

2.4.2函数的作用

def 函数名():

​ 过程代码

函数的作用就是把这个过程，可具有相同类别的内容，规格统一放在一个代码块中，并根据函数的执行过程，或业务的执行流程来调用不同的函数或增减函数

2.4.3函数的定义

在Python中，定义一个函数我们采用顶格的def关键字来定义一个函数，一个函数结构一般包含以下内容：

def 函数名(参数1，参数2，…..):

​ 过程代码

​ return 返回值

以上为最大函数体的结构组成

以下为最小函数体的结构组成

def 函数名():

​ 过程代码

返回值

一个函数可以至少有一个返回值，可以认为指定任务类型的返回数据，如果没有人为指定，那么返回值为none，返回值的个数，可以一个，也可以是多个，函数的返回除了是常见的数据类型以外，还可以是另一个函数

在以下几个方面，return是等价的

1、没有return

2、return（只有return，后面不更任务的变量和结果）

3、return None

在函数中，可以使用return将里面的结果返回出来，代码一旦运行到return那么函数就会结束，return后面的代码就不会执行

函数定义和书写的注意事项：

在一个Python工程中，应该保证每个函数的名字是唯一，函数体就是这个函数执行的一系列的操作。操作可能只有一行代码，也可能是多行。

一个函数原则上只能做一件事，同时Python编码规范定义，一个函数的函数体最多不能超过20行代码，如果超过了，说明这个函数做了不止一件事。如果超过，应该将这个函数拆解为多个小函数。这儿也是说函数体可以调用其他函数。
