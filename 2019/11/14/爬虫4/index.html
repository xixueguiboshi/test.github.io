第三章：python的文件读写和sqlite操作

3.1Python的文件读写

文件操作一般通过内置的open函数完成，这个函数很像C语言中的fopen，通过open函数我们可以获得一个file object

python 提供的另一个内置函数read():用于读取file object后的内容操作

python 提供的另一个内置函数write():用于对内容进行编辑(读写)操作

在文件操作完成以后，一定要关闭close()

1.open()

使用方法：open(‘文件的路径’)

文件的路径：相对路径和绝对路径

file = open (“demo.txt”)

3.1.1文件的读操作

open()

try:

​ read()

finally:

​ close()

以上为读操作的步骤和要求，注意事项：

1、不能在open时，就try语句，因为文件如果一打开就出错，close就不知道怎么办了。

2、使用open打开文件后，一定要记得调用close方法进行关闭，如果不关闭，可能造成文件被占用而无法被一个需要open操作所使用

3、文件read()是可以设置读权限：如只读，如强制为二进制等等

4、open方法也可以通过设置权限来完成相关的操作读写权限等等

3.1.2Python文件的写操作

读：read()

写：write()

open()

try:

​ write()

finally:

​ close()

注意事项：

1.打开文件，应该具有写模式的参数打开文件

2.参数不同，写入文件的效果不同

3.writelines()写入多行在性能上，要比write()一次性写入要高

3.2 sqlite(Python操作数据库)

数据库：关系型数据库-DBMS

sqlite此数据库是系统内置，目前发布到sqlite3.x

如果Python需要控制或操作sqlite则需要使用对应的库(lib)

在数据库，一个完整的数据库操作基本上涵盖以下几个部分，且可以根据实际情况进行裁剪：

3.2.1数据库创建

create database xxx

在Python中，2.5以后的版本已经内置了相关支持库，我们只需要引入就可以，不需要下载

在连接加载库文件时，一定注意文件名是否写正确如果不正确会产生新建操作

3.2.2数据库表的创建

create table tablename(clo type,clo type)

如create table user(id int,primary key)

在Python中如何执行sql?

1、加载数据库文件

2、通过加载的连接对象，创建一个游标：cursor()

3、通过游标对象，去执行sql语句：execute()

4、提交执行事物：commit()

5、关闭连接，将开的库文件释放掉，以便其他程序调用

3.2.3数据库表的操作(select：检索、查询操作)

在执行查询语句后，Python会返回一个结果，循环器，这里面包含查询所需要获得记录和相关信息，我们可以通过循环读取的方式，来读取数据。

语法：

select * from tablename where 条件

fetchone()

fetchall()

以上两个函数一定要结合游标来使用才能出结果

除了以上比较直接获得结果内容以外，还要以通过循环游标的循环器来获得每行数据，此时每行数据为元组类型

3.2.4数据库表的操作(delete：删除)

drop 权限很大慎用

3.2.5数据库表的操作(update：更新)

update tablename set 字段名=新值 where 字段名=名字

3.2.6数据库表的操作(insert：插入)

插入值的语法为：

insert into tablename values(1,2,3,4)#全匹配插入值

insert into tablename (列表1，列表2) values(值1，值2)

这里面有批量插入函数executemany(),支持通过Python批量插入
