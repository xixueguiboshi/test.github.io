第二章：Python基础

Python开发环境的搭建，官方网站“www.pthon.org”

2.1 python的安装与运行

由于历史原因，目前市面上流通的Python有两个版本，Python2.x 和Python3.x，如何区别代码2还是代码3写的，看或写：

print(‘’);//python3

print’’;//python2

为什么需要区别版本，因为两个版本相互不兼容的

Python官方宣布过，在未来Python的发展主要以3为主，Python3会新增的功能，而2只做过修复bug，不会新增功能

2.1.1安装

Python3.7.1(可以去官网下最新版，差异不大)

windows:

首先先查看系统环境

32位对应32位的环境

64位对应64位的环境（兼容安装32位）

Mac os:

Homebrew安装：brew install python3

在Macos中，已经内置Python2

如果无法完成homebrew安装，可以到官网下载macos对应的pkg安装包进行安装

Linux:

Python –version 查看当前系统版本，如果版本低或没有安装，或以通过以下命令完成：

sudo apt-get update

sudo apt-get install python3.7 python3-dev python3.pip libxml2-dev libxslt1-dev zlib1g-dev libffi-dev libssl-dev

面向脚本的开发工具就特别多

两个内置函数

2.3.1整数、浮点数、变量

A：整数和浮点数

Python中的整数和数学中定义的整数是一样的。

Python中的浮点数和数学中定义的小数是一样的。

在Python中，我们可以通过print打印一个整数或浮点数

如：

print(1234) print(1.234)

整数的加减乘除运算可以在print中直接进行，同时还支持通过括号的方式来改变运算的优先级

如：

print(1-10)

print(3+2-5*0)

print((3+2-5)*1)

在Python中做除法运算会得到浮点结果

如

print(5/5)

结果是：1.0

在浮点运算中，如0.3+0.1结果不是0.4，而是很长一段浮点值，这不是Python的问题，这是由于浮点数的存储机制导致的。

注意：不应该直接Python进行精确计算，但是进行日常精度不高的四则运算是没有问题的。

B:变量

变量是可以变的，可以理解为一个存放其他数据的盒子，使用变量可以减少重复的输入。

如：

len=10

width=15

height=20

D=width*len

T=D*height

print(D)

print(T)

一个合法的变量为等号左边为变量名，右边为变量值，其中等号是赋值的意思。

如

name=”zhiliaotang” 意思就是变量名为name 值为”zhiliaotang”

补充：变量的类型完全由赋值结果来确定结果是整形就是整形是字符串就是字符串。

Demo 完成一个简单的计算器小程序，实现简单的四则运算即可。要求用户输入值程序接收到值在进行计算，并打印结果。

cmd下直接运行Python脚本的方法

如脚本名为：abc.py

cmd》python abc.py

2.3.2字符串

在Python中字符串是一种比较常见的数据类型，任何被单引号或双引号括起来的内容，都可以被认为是字符串，字符串也可以赋值给

string_1=’我是谁’#字符串可以接受中文或英文或其他语言的字符内容

string_2=’im zhi’

string_3=’’#空字符串

string_4=’ ‘#空格字符串

string_5=’a’#只有一个字母

string_6=””#采用双引号的字符串定义

从上面的行码可以看出，字符串的内容可以是中文、英文、数字、空格以及其他字符组成。

字符串形式的数字和普通数字是不一样的如：

int=123

string=”123”

以上两个变量的结果不一样的

2、列表(list)

数据类型的表现形式为：list = []

列表在Python中是一种容器类型，由方括号和方括号中的数据组成。里面的数据可以是整数、浮点数、字符串，也可以是另一个列表或其他数据结构

列表中的每一项叫作列表的一个元素，每一个元素之间使用逗号隔开。

如：

List_1=[1,2,3,4,5,6]#列表中有6个元素，全都是数字

List_2=[‘abc’ , ‘x’]

List_3=[]#空列表

List_4=[‘123’,123,3.2,[123,’as’]]#由多种元素组成的复合列表

3、元组(tuple)

数据类型的表现形式为：tuple=()

元组也是Python中的一种容器类型，由小括号和小括号括起来的数据构成，外型和列表非常像，只不过列表为方括号，元组为小括号，”元组”中的”元”和”二元一次方程”中的元是一个意思，”组”就是组合的意思。

如：

tuple1=(1,2,3,4,5,6)#元组中有6个元素，都为数字

tuple1=(‘123’,’sss’)

tuple1=()

tuple1=(‘123’,123,3.2,[123,’as’],(1,2,3))

元组支持类型>列表>字符串>整数

元组和列表的区别：

列表生成以后，还可以往里面继续添加数据，也可以从里面删除数据

元组生成以后，就不能修改以下几种类型的数据：整数，浮点数，字符，另一个元组。里面是列表的除外

2.3.3数据的读取

字符串、元组、列表三种数据结构的读取方式完全一样

1指定下标

在大多数的编程语言里面，下标都是从0开始的，Python也不例外，，也是从第0个元素开始，第0个指的是元素最左边的那个。

string_name=’yan’

在字符中，第0个字符是”y”

list=[‘a’,’b’,’c’] 在列表中第0个元素是”a”

tuple=(1,2,3)在元组中第0个元素是1

在以上三种类型的数据中,想要任何一个元素的内容，都可以直接使用以下表达式：

变量名[下标] 来获取对应下标的内容

0-字符长度-1，如字符串长度为8个字符，则范围为0-7

实际取值范围是-8-0-7

下标如果为正值则正向取值，如

name=’yan’

name[0]=>’y’

name[-1]=>’n’

2切片操作

字符串通过切片以后，结果还是字符串，列表切片以后还是列表，元组切片以后还是元组

为什么要切片，需要获取某个范围的内容时，需要用到切片操作

切片格式为：

变量名[开始位置的下标：结束位置的下标：步长]

其中，开始位置的下标，结束位置的下标，步长都可以部分省略

这三个参数都是数字

步长：

步长正值从左到右

负值从右到左

3、拼接和修改

字符串的拼接

string1=’123’

string2=’456’

string3=string1+string2

修改内容

字符串的修改

字符串重新赋值即可修改

name=’123’

name=’456’

元组和列表的修改采用下标完成

list=[1,2,3,4]

list[1]=10

list.append(“”)可以向列表添加元素

#元组是不可变容器，列表为可变容器，所以append()只能作用于可变容器

2.3.4字典与集合

1、字典

字典使用大括号括起来的键(key)值(value)对，每个键值对之间使用英文逗号分隔，每个key-value之间使用英文的冒号分隔，如

dict={‘name’:’yan’,’work’:’java’}

key:中文、英文或数据，但是不能重复

value: 可以是任意字符串、数字、列表、元组或另一个字典，value可以重复

字典的取值：

变量名[key]

变量名.get(key)

2、集合

集合就是没有key的，由大括号括起来的内容，而内容可以是不同的数据类型，如：set={‘1’,2,3,’c’}

集合里面的元素是不能重复的，集合也表现出了无序特征

集合最大的应用之一就是去重操作，列如，把一个带有重复元素的列表转为集合，再转回列表，那么重复的内容元素只会保留一个，把列表转换为集合需要使用set()函数，把集合转换为列表用list()函数

list1=[3,1,2,5]

union=list(set(list1))
