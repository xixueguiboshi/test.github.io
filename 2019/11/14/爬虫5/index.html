4.1关键字的学习

4.1.1 requests 库

该库是一个第三方，主要的作用就是基于开源的AL开发协议的HTTP库，比urllib更加方便，可以节约大量时间。

import requests

创建请求：

普通的HTTP请求有两种方式：get/post

r=request.get(‘http://www.baidu.com')#等于在浏览器中输入url的方式

r=request.post(‘’)#等于是类似from表单或隐形的请求，这个方式更安全，但是也很多阻碍

通过get/post请求以后，会返回一个response对象，里面有我们需要的信息除了上面两个函数还有put() delete() head() options() ,结合这些函数，可以帮助我们进行更复杂的操作反爬虫机制

4.1.2 url中传递参数

要想在url中查询数据，我们可以在url后加上？及键值对如：

http://www.baidu.com?Kw=zhiliaotang

4.2LXML

lxml是python中的一个非常高效的解析库，支持xml和HTML，对于内容，他支持xpath解析方式

cp el

cp xpath(xml path language),他是一门在xml文档中查找信息的语言，他最初是用来做xml文档搜寻工作，现在同样的使用于HTML的内容或文档搜索。

xpath在爬虫中主要依赖lxml库来完成节点的定位，以方便我们获得节点中的数据

4.2.1xpath常见规则

表达式：

nodename:选取当前节点的所有子节点，如body/dev

/：从当前节点直接选取子节点，如：/html/body

//:从当前节点直接选取子孙节点

.:选取当前节点

..:选取当前节点的上一级节点等于是选取父节点

@：选取属性，如：

@name

*:通配符，选取所有元素节点与元素名

@*：选取所有的属性

[@atrrib]选取所有属性的所有元素

[tag]选取所有具有指定元素的直接子节点

[tag=’text’]选取所有指定元素并且文本内容是text节点
