python爬虫数据可视化

准备工作：

1、可以进行爬虫的脚本，且脚本支持数据存储（存储容器建议支持sqlite/mysql/sqlserver）

2、要有一个有数据的数据库文件或系统（sqlite3库本名：xxx.db）

3、具备HTML+CSS+JS的基础应用能力

​ （1）html:内容

​ （2）css:布局和样式

​ （3）JS:交互和事件

4、开发环境：

​ （1）IDE:HbuilderX

​ （2）Sqlite 5.2管理工具

目标：

通过上一章节中，爬虫爬回来的数据（数据是存放于sqlit3DB文件中），然后通过JS的异步加载方式，加载DB文件，然后从DB中用数据库语法查询出所需要的数据，然后再通过HTML+css将他们呈现在网页上（完成上述功能，需要一个支持HTTP协议的服务器）

要实现数据从sqlite在读取过来，需要的技术？

异步加载：Ajax

核心是js对象XMLHttpRequest，他是一种支持异步请求的技术
